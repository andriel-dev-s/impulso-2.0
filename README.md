# 🚀 Impulso Local

O **Impulso Local** é uma plataforma criada para ajudar negócios locais a crescerem na internet, atraindo mais clientes através de marketing digital e presença online.

---

## 🌍 Sobre o projeto

Muitos negócios locais não têm presença forte na internet.
O Impulso Local resolve isso oferecendo ferramentas simples para:

* 📍 Aparecer no Google (Google Maps)
* 📱 Crescer no Instagram
* 📢 Divulgar o negócio localmente
* 📊 Acompanhar informações básicas do negócio

---

## 🧠 Objetivo

Criar uma plataforma acessível para pequenos empreendedores conseguirem:

* Mais visibilidade
* Mais clientes
* Mais vendas

---

## 💻 Funcionalidades atuais

* ✅ Landing page estilo startup
* ✅ Cadastro de empresas
* ✅ Painel com empresas cadastradas
* ✅ Armazenamento local (localStorage)

## 🚀 Como rodar o projeto

1. Baixe ou clone o repositório:

```
git clone https://github.com/seuusuario/impulso-local.git
```

2. Abra o arquivo:

```
index.html
```

3. Pronto! O projeto já estará funcionando no navegador.

---

## 🌐 Deploy (GitHub Pages)

Você pode colocar o projeto online:

1. Vá em **Configurações**
2. Clique em **Páginas**
3. Selecione a branch `main`
4. Salve

Seu site ficará disponível em:

```
https://seuusuario.github.io/impulso-local
```
